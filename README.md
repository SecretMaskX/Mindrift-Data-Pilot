AI-Powered Data Engineering & LLM Training Pipeline

This repository demonstrates an end-to-end data pipeline designed for high-scale web scraping and automated dataset enrichment for Large Language Models (LLMs).
🚀 Key Features

    Dynamic Scraping: Extraction of structured data from hierarchical sources like Hacker News.

    AI-Driven Enrichment: Automated data labeling and analysis using Groq (Llama-3.3) to create high-quality training sets.

    Technical Troubleshooting: Successfully optimized performance by migrating from Google Gemini to Groq API to bypass rate-limiting (429 errors).

    Security-First Approach: Implemented environment variables for sensitive API credentials.

🛠 Tech Stack

    Python (Pandas, BeautifulSoup, OS).

    Groq Cloud API (Inference at scale).

    Arch Linux (EndeavourOS) (Development environment).
